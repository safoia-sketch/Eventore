# Eventore

Eventore is a full-stack event-management and ticket-booking platform.

## Technology

### Frontend

- React
- Vite
- React Router
- Bootstrap
- Custom CSS

### Backend

- Node.js
- Express
- PostgreSQL
- Session authentication
- bcrypt
- express-validator
- QR-code generation

## Main roles

- Attendee
- Organiser
- Check-in staff
- Administrator

## Main features

- Event discovery and filtering
- Event creation and approval
- Ticket-type management
- Safe ticket booking
- Digital tickets
- QR-code check-in
- Booking cancellation
- Organiser reports
